# ClauDex 🧡🖤

> Claude Code plugin that makes Claude and Codex pair program — Claude writes, Codex reviews, ship on consensus.

The wedge: OpenAI open-sourced [codex-plugin-cc](https://github.com/openai/codex-plugin-cc)
(GitHub trending, Jul 8 2026), teaching thousands of devs the `/plugin marketplace add`
install motion. ClauDex rides the exact same motion and turns the rivalry into a QA
process — cross-model agreement is the strongest free review signal there is, and
disagreement is exactly where a human should look. Every run signs off with
`built with love by ClauDex 🧡🖤` and commits carry both AI co-authors — a growth
loop embedded in other people's git history.

Code is public: [hamza-ali-shahjahan/claudex](https://github.com/hamza-ali-shahjahan/claudex).
Distribution is the GitHub plugin marketplace, **not npm** — `claudex` and
`claudex-cli` are already taken there (see `learnings.md`).
