# Repolish

> Make any repo's first impression look premium **and** honest — in one command.

**Working name (not locked).** The wedge vs. every other README generator: a
**honesty pass** that flags overclaims ("production-ready" with no tests, "tested on
Windows" with no CI, fake benchmarks) so the polish never becomes hype — plus a
**real** demo-GIF recording instead of a faked one.

Code is private (this is the always-public factory repo). The implementation lives
in a local-only folder registered in the gitignored `code-paths.local.json` under
slug `repolish`. Nothing is published until the name is locked.
